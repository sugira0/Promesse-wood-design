# Logo Files

Please save your logo images in this folder with the following names:

1. `logo-full.png` - The full logo with tree icon and "Promesse Wood Design" text (use the first brown/gold version)
2. `logo-icon.png` - Just the tree circle icon (for favicon and mobile)
3. `logo-light.png` - The white/cream version of the logo (use the second version)

## Recommended sizes:
- Full logo: 400x600px (maintains aspect ratio)
- Icon: 512x512px (square)
- Favicon: 32x32px, 180x180px (will be generated from icon)

The HTML files have been updated to reference these logo files.
